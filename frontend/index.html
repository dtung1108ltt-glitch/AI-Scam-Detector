Endpoints
Base URL: https://api.scamshield.vn/v1
POST/checkPhân tích nội dung scam
GET/historyLấy lịch sử quét
GET/statsThống kê tổng hợp
POST/reportBáo cáo scam mới
Ví dụ sử dụng
// Gọi API ScamShield const response = await fetch('https://api.scamshield.vn/v1/check', { method: 'POST', headers: { 'Authorization': 'Bearer YOUR_API_KEY', 'Content-Type': 'application/json' }, body: JSON.stringify({ message: 'https://bnb-airdrop-claim.xyz', platform: 'crypto' }) }); const data = await response.json(); // { result: "SCAM", confidence: 0.97, reason: "..." } console.log(data.result); // "SCAM"
